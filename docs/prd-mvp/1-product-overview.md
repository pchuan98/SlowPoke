# 1. 产品概述

### 1.1 产品定位

**SlowPoke** 是一个以个人为中心的 TODO 管理工具，基于以下核心理念：

- ✅ **个人 TODO 管理工具**（核心身份）
- ✅ **本地优先存储**（数据主权）
- ✅ **Markdown 为载体**（方便迁移）
- ✅ **可扩展平台**（类似 MCP 模型）
- ✅ **记录为王**（辅助完成多维度交叉项目）

### 1.2 MVP 范围

MVP 版本聚焦于**核心个人 TODO 管理能力**，提供：

1. 基础 TODO CRUD（创建、读取、更新、删除）
2. 四种展示视图（List、Table、Kanban、Timeline）
3. Markdown 编辑器（元数据和内容分离）
4. 项目概念（项目即 TODO）
5. 灵活的过滤筛选
6. 单用户 Web 应用
7. 数据库 + Markdown 双存储架构

**MVP 不包含：**

- ❌ 多用户系统
- ❌ 复杂权限管理
- ❌ TODO 关系运算（依赖、阻塞等）
- ❌ 工作上下文容器（资料、问题、思考）
- ❌ AI 功能（标题生成等）
- ❌ 多端同步
- ❌ 数据加密

### 1.3 用户场景

**主要用户**: 个人开发者（项目创建者自己）

**核心使用场景**:

1. 从日常生活中提出一个 TODO，添加标题和描述
2. 将 TODO 归属到某个项目下
3. 使用 Markdown 记录详细内容
4. 在不同视图间切换查看 TODO（List、Table、Kanban、Timeline）
5. 通过过滤条件筛选 TODO（按项目、状态、优先级等）
6. 选择性通过分享 URL 展示部分 TODO 给他人
