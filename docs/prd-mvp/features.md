# 未来功能规划

以下功能在 P0 版本中不实现，作为后续版本的功能储备。

## 视图功能

1. Table 视图 - 表格展示，支持列排序和多列筛选
2. Kanban 视图 - 看板展示，支持拖拽改变状态
3. Timeline 视图 - 时间线展示，按时间顺序排列

## 数据管理

4. 项目管理 - 项目作为特殊 TODO，存储在独立目录，支持 TODO 关联
5. 过滤与筛选 - 多维度过滤（状态、优先级、项目、时间范围、关键词搜索）
6. 数据库重建 - 从 Markdown 文件重建数据库索引

## 协作功能

7. 分享功能 - 生成只读分享链接，展示过滤后的 TODO 列表

## 扩展能力

8. 多用户系统 - 用户管理、权限控制
9. TODO 关系运算 - 依赖关系、阻塞关系
10. 工作上下文容器 - 关联资料、问题、思考
11. AI 功能 - 标题生成、内容摘要、智能分类
12. 多端同步 - 云端同步、多设备访问
13. 数据加密 - 端到端加密、敏感数据保护
