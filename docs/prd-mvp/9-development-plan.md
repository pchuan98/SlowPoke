### 9.3 技术风险与应对

| 风险                  | 影响 | 概率 | 应对措施               |
| --------------------- | ---- | ---- | ---------------------- |
| C# AOT 兼容性问题     | 高   | 中   | 先用 JIT，后续迁移 AOT |
| Markdown 编辑器性能差 | 中   | 低   | 切换为其他编辑器       |
| Kanban 拖拽库 Bug     | 中   | 低   | 降级为手动选择状态     |
| SQLite 并发写入冲突   | 低   | 低   | 单用户场景无影响       |
